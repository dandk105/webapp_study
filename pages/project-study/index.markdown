このページは、このプロジェクトの進め方についての学習を段階的に行うことができます

全体の構成は以下のようになっています。

- プロジェクト計画書
- ソフトウェアアーキテクチャ図
- 概要設計書
- 詳細設計書

それぞれのコンテンツは以下のようなフォーマットで作成されています
1. 学習の対象者
    1. 前提条件
2. ゴール
3. ドキュメントのスコープ
4. 何を教えようとしているのか
5. ドキュメントを読み込むことで得られるメリット、デメリット
